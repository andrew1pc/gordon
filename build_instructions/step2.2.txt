Step 2.2.2: Implement Rate of Change (ROC)
===================================================================

Implement the calculate_roc method in indicators/momentum.py. ROC measures momentum:

Formula: ROC = ((current_price - price_n_periods_ago) / price_n_periods_ago) * 100

The method should:
- Calculate ROC for specified period (typically 20 and 50 days)
- Add column 'roc_{period}' to DataFrame (e.g., 'roc_20', 'roc_50')
- Handle edge cases where data length < period
- Positive ROC = upward momentum, negative = downward
- Return DataFrame with ROC column added

Include example showing ROC > 20% is strong momentum, ROC < -20% is weak.
